Estructura Básica de HTML

Todo documento HTML sigue una estructura básica:


<!DOCTYPE html>
<html>
    <head>
        <title>Título de la Página</title>
    </head>
    <body>
        <h1>Hola Mundo</h1>
    </body>
</html>

Etiquetas Básicas

Algunas etiquetas esenciales son:

  • <p>: Párrafos
  • <a>: Enlaces
  • <img>: Imágenes
  • <ul> y <li>: Listas

Estilos con CSS

Podemos modificar la apariencia con CSS. Por ejemplo:


body {
    font-family: Arial, sans-serif;
    background-color: #f4f4f4;
}
